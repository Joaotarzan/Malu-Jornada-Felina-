# 🐱 MaluVerso

Um joguinho 2D feito em JavaScript onde você controla a minha gata, Malu, correndo por vários mundos, pegando peixes e desviando dos perigos.

Fiz esse projeto como uma forma de aprender mais sobre canvas e também como uma homenagem pra ela.

---

## 🎮 Como jogar

* Pule com `Espaço`, `↑` ou `W`
* Abaixe com `↓` ou `S`
* Também dá pra clicar na tela pra pular

Objetivo: fazer pontos coletando peixes e sobreviver o máximo possível.

---

## 🗺️ Níveis

O jogo tem vários mapas com temas diferentes, e a dificuldade vai aumentando conforme você avança.

Cada fase tem:

* mais velocidade
* mais inimigos
* mais coisa acontecendo na tela

---

## ✨ O que tem no jogo

* Sistema de fases com desbloqueio
* Pontuação e recorde por nível
* Mapa interativo
* Efeitos visuais simples (partículas, animações, etc)
* Salvamento automático no navegador

---

## 🚀 Como rodar

Só abrir o `index.html` no navegador.

Se quiser rodar com servidor:

```bash
npx serve
```

---

## 🛠️ Feito com

* HTML
* CSS
* JavaScript (canvas)

---

## 💜 Sobre a Malu

A Malu é minha gata, e esse jogo é basicamente uma versão dela como protagonista de um joguinho.
